# GitHub Features

1. Repositories – Store project files and version history.
2. Branches – Work on new features without affecting main code.
3. Pull Requests – Propose and review changes before merging.
4. Issues – Track bugs and tasks.
5. GitHub Actions – Automate workflows and CI/CD.
6. Projects – Manage tasks using boards.
7. Wiki – Create documentation pages.
8. Releases – Publish project versions.
